## 🚀 Deploy

### Vercel (recomendado)
1. Conecta este repo en [vercel.com](https://vercel.com)
2. Deploy automático con `vercel.json` incluido
3. Conecta tu dominio en **Settings → Domains**

### Netlify
1. Arrastra la carpeta en [drop.netlify.com](https://drop.netlify.com)
2. URL pública instantánea

## ✏️ Cómo Actualizar Precios o Servicios

1. Abre `index.html` en GitHub (clic en el archivo → ✏️ lápiz)
2. Usa `Ctrl+F` para buscar el servicio a modificar
3. Cambia el precio en la línea `<div class="price">$XXX.XXX</div>`
4. Clic en **"Commit changes"**
5. Vercel detecta el cambio y actualiza el sitio automáticamente ✅
